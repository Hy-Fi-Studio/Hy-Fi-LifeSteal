# Hy-Fi-LifeSteal
Hy-Fi LifeSteal brings the popular LifeSteal gamemode to Hytale multiplayer. Players gain hearts by killing others and lose hearts on death. Includes soft wipes, team support, custom crafting options, and scalable late-game progression. Ideal for survival servers and PvP networks.
